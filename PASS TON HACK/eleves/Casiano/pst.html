Dans Le Repaire – Partie 1, nous avons commencé par analyser l’image fournie. Grâce à l’observation des éléments visuels et à l’utilisation de Google Maps, nous avons identifié le Panthéon. Cette localisation nous a permis d’obtenir le FLAG demandé.

Écho de l’Abîme – Décodage sonar Morse

Démarche de résolution

Pour Écho de l’Abîme – Décodage sonar Morse, nous avons étudié le fonctionnement du code Morse (points et tirets). Après plusieurs recherches sur des forums et quelques erreurs de compréhension, nous avons corrigé notre méthode. Nous avons utilisé un dictionnaire Morse et un programme Python pour traduire automatiquement le message en texte lisible.

Survivant Crescendo

Après de nombreux tests dans la console du navigateur et des recherches complémentaires, nous avons réussi à devenir invincibles, vaincre le boss final et récupérer le code.

Écho de l’Abîme – Reconstruction de fragments

Démarche de résolution

Enfin, pour Écho de l’Abîme – Reconstruction de fragments, nous avons identifié que le message était composé de fragments numérotés. Nous avons donc écrit un script qui découpe les fragments, les trie selon leur numéro, puis reconstitue le message final.
